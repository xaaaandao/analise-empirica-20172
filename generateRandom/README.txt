Gerador de valores aleatórios definindo o intervalo e a quantidade pelo usuário
1 - Para compilar use o seguinte comando: gcc main.c -o main
2 - Para executa use o seguinte comando: ./main minValue maxValue count
3 - O minValue e maxValue deve ser substituído pelo valores de intervalo na qual deverá ser gerado os valores.
4 - O count deve ser substituído pela quantidade de valores que o usuário deseje gerar.
